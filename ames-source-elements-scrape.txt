Ames product page scrape mapping for SKU, MPN, Name, Description, Price, and Image URLs

Source page sample: TapeTech U-Cup Wiper 729009

Field: Name
- CSS selector: h1.product-details-full-content-header-title[itemprop="name"]
- XPath: //h1[contains(@class, "product-details-full-content-header-title") and @itemprop="name"]
- Sample extracted value: TapeTech U-Cup Wiper 729009

Field: MPN
- CSS selector: div.mpn
- XPath: //div[contains(@class, "mpn")]
- Extraction rule: strip leading text, e.g. "MPN : 729009" -> "729009"
- Sample extracted value: 729009

Field: SKU
- CSS selector: span.product-line-sku-value[itemprop="sku"]
- XPath: //span[contains(@class, "product-line-sku-value") and @itemprop="sku"]
- Extraction rule: trim whitespace from the text content
- Sample extracted value: 14972

Field: Price
- Preferred CSS selector: span.product-views-price-exact[itemprop="offers"] meta[itemprop="price"]
- Preferred XPath: //span[contains(@class, "product-views-price-exact")]/meta[@itemprop="price"]/@content
- Fallback CSS selector: span.product-views-price-lead[data-rate]
- Fallback XPath: //span[contains(@class, "product-views-price-lead") and @data-rate]/@data-rate
- Sample extracted value: 15.19

Field: Description
- CSS selector: div.product-details-information-tab-content-panel[itemprop="description"]
- XPath: //div[contains(@class, "product-details-information-tab-content-panel") and @itemprop="description"]
- Secondary selector: div#product-details-information-tab-content-container-0[data-type="information-content-text"]
- Extraction rule: extract the visible text from within the description panel, then normalize whitespace
- Sample extracted value: TapeTech U-Cup Wiper for 76TT part #729009

Field: Image URLs
- CSS selector: div.product-details-image-gallery img[itemprop="image"]
- XPath: //div[contains(@class, "product-details-image-gallery")]//img[@itemprop="image"]/@src
- Capture all distinct src values for the gallery images
- Sample extracted URLs:
  - https://www.shopamestools.com/Item_Images_AMES/14972_TapeTechUCupWiper729009_002.jpg?resizeid=16&resizeh=1200&resizew=1200
  - https://www.shopamestools.com/Item_Images_AMES/14972_TapeTechUCupWiper729009_001.jpg?resizeid=16&resizeh=1200&resizew=1200

Notes:
- The page uses server-side templating and itemprop attributes for structured data.
- The most reliable extraction fields are itemprop attributes for name, sku, and price.
- MPN is rendered in plain text inside a container with class "mpn".
- Description is stored inside the product information tab panel marked with itemprop="description".
- Image URLs are available as direct src attributes on the gallery image elements.
