Columbia Tool Sets — Official Specifications & Details (2026)
Overview

Columbia Taping Tools offers a family of drywall tool sets designed to take the uncertainty out of choosing taping and finishing tools. Each set bundles tools needed for a particular level of automation (automatic taping, semi‑automatic taping or smoothing) and packages them in a dedicated case for transport and protection. The official item numbers and contents of these kits are documented on Columbia’s website and catalogues. The information below compiles the latest specifications (as of 2026) and cites Columbia’s official product pages and catalogue.

Tactical Set (TS)

The Tactical Set is Columbia’s flagship automatic taping starter kit. It is marketed as a complete package for professionals who want to move from hand‑taping to automatic taping. According to Columbia’s product page, the set includes every taping tool needed to completely finish a house and comes in a durable road case. It emphasizes ease of transition to automatic tools and warns that Columbia’s automatic boxes do not fit inside the road case.

Official contents

The Tactical Set (item No. TS) comes with the following tools:

Road Case (protective storage case).
Automatic Taper — Columbia’s standard automatic taper used for applying tape and mud to seams.
10 in & 12 in Flat Finish Boxes — for finishing flat seams; used with the Matrix handle.
Matrix Handle — adjustable handle for flat boxes.
Tomahawk Smoothing Blades: 7 in and 24 in blades.
8 in Throttle Box — a corner finisher/angle box with lever‑actuated mud control.
3 in Angle Head — finishes internal corners after rolling tape.
Hot Mud Pump with Box Filler & Gooseneck — pumps mud into taper and boxes.
Corner Roller — embeds tape into inside corners.
AHA (Angle Head Adapter) — connects angle heads to handles.
3 in Nail Spotter — fills nail/screw dimples.
Columbia One Handle & Columbia One EXT — universal handles compatible with multiple tools.
Sander Head & Twist‑Lock Handle — hand sander head with adjustable handle.
Tomalock Adapter — connector for Tomahawk blades.
Repair kits: AH‑RK (Angle‑head), FFBR9‑10, FFBR9‑12, FFBR7‑7A (flat‑box repair kits) and CTR‑1 (corner roller repair kit).

These components make the Tactical Set the most comprehensive of the Columbia kits, aimed at contractors who need a full automatic solution.

Predator Tactical Set (PTS)

The Predator Tactical Set is similar to the Tactical Set but upgrades the standard automatic taper to Columbia’s Predator carbon‑fiber automatic taper. Columbia describes the set as a way to move to automatic taping with this lightweight, stronger taper. As with the Tactical Set, the tools are packed in a rugged road case, but Columbia notes that automatic boxes do not fit inside the case.

Official contents

The Predator Tactical Set (item No. PTS) includes the following components:

Road Case.
Predator Automatic Taper (carbon‑fiber), replacing the standard taper.
10 in & 12 in Flat Boxes with Matrix Handle.
7 in & 24 in Tomahawk Smoothing Blades.
8 in Throttle Box (Angle Box).
3 in Angle Head.
Hot Mud Pump with Box Filler & Gooseneck.
Corner Roller and AHA (Angle‑Head Adapter).
3 in Nail Spotter.
Columbia One Handle & Columbia One EXT.
Sander Head & Twist‑Lock Handle.
Tomalock Adapter.
Repair kits: AH‑RK, FFBR9‑10, FFBR9‑12, FFBR7‑7A, CTR‑1.

The only difference from the standard Tactical Set is the Predator carbon‑fiber taper, which is lighter and more durable. Otherwise the included tools mirror the Tactical Set.

Commando Set (SACS)

Columbia’s Commando Set provides an introduction to semi‑automatic drywall tools. The product page says it is ideal for part‑time finishers or those looking to bridge the gap between hand taping and full automation. It supplies the essential semi‑automatic tools in a dedicated case.

Official contents

The Commando Set (item No. SACS) comes with:

Semi‑Automatic Case (protective storage).
2.5 in Standard Flusher (2.5SF) — internal corner flusher for finishing narrow angles.
3 in Standard Flusher (3SF) — larger flusher for finishing standard inside corners.
42 in Compound Mud Tube (CMT42) — holds compound and feeds it to applicator heads.
Columbia One Extendible Handle (C1HEXT) — extendible handle for flushers.
Columbia One Fixed Handle (C1H) — standard handle for applicators.
Angle‑Head Adapter (AHA).
Flat Applicator Head (CFLT) — attaches to mud tube for applying compound to flats.
Two‑Way Internal Applicator (ICATW) — internal corner applicator that applies compound to both sides of a corner.
Corner Roller Head (CR) — used to embed tape tightly into internal corners.

Unlike the Tactical sets, the Commando kit focuses on semi‑automatic tools for corner finishing and compound application without including a full automatic taper.

Warrior Set (TWS)

The Warrior Set is designed for contractors transitioning into smoothing blade work. Columbia notes that this kit provides a range of Tomahawk smoothing blade sizes for wiping tapes, coating walls and ceilings and general finishing. It comes with a “gun case”‑style carrying case to protect the tools.

Official contents

The Warrior Set (item No. TWS) comprises:

Gun Case (protective case).
Tomahawk Smoothing Blades: 7 in, 12 in, 18 in and 32 in.
Tomalock Adapter — converts Tomahawk blades for use with handles.
Columbia One EXT Handle — extendible handle for smoothing blades.

This set focuses exclusively on Tomahawk smoothing blades and does not include tapers, flat boxes or pumps.

Sabre Set (SBSET)

The Sabre Set is another smoothing blade kit but uses Columbia’s Sabre blades instead of the Tomahawk line. The product description explains that it provides multiple blade sizes for wiping tapes, coating walls and ceilings and includes a case for easy transport. The Sabre blades are designed to simplify sanding and finishing.

Official contents

The Sabre Set (item No. SBSET) consists of:

Sabre Case.
Sabre Smoothing Blades: 10 in, 16 in, 24 in and 32 in.
Sabre Claw Adaptor — attaches blades to handle.
Sabre Pole — telescopic handle for the blades.

Unlike the Tomahawk kit, the Sabre Set uses a different blade profile (Sabre) and the included pole rather than the Columbia One handle.

Fat Boy Smoothing Blade Set (FSBSET)

Announced under Columbia’s New Releases section, the Fat Boy Smoothing Blade Set is a complete finishing kit aimed at professional drywall finishers. The product page describes it as an all‑in‑one system covering compound application, skimming and detail work. The Fat Boy blades use an ergonomic grip and replaceable stainless‑steel blades, and the kit pairs them with a mud roller and telescopic handle to reduce fatigue.

Official contents

The Fat Boy Smoothing Blade Set (item No. FSBSET) includes:

Semi‑Automatic Case (TCS) — protective case for the kit.
Fat Boy Smoothing Blades: 7 in, 14 in, 18 in and 32 in.
Fat Boy Smoothing Blade Claw (FSBC) — adapter/handle for the blades.
Fiberglass Telescopic Extension Handle (4–6 ft) (FTH4‑6) — adjustable handle.
9 × 18 mm 2K‑Handle Mud Roller (2KMR9) — roller for mud application.
9‑in‑1 Comfort Grip Multi‑Tool (91CGMT) — multipurpose hand tool.
3‑Way 3‑Piece Joint Knife Set (3WJKBP) — set of joint knives for detail work.

This kit extends Columbia’s smoothing blade offerings by adding additional hand tools (multi‑tool, joint knife set) and a mud roller, making it a comprehensive finishing system.

Conclusion

Columbia’s official product pages and catalogue confirm that only six tool sets are currently offered: the Tactical Set (TS), Predator Tactical Set (PTS), Commando Set (SACS), Warrior Set (TWS), Sabre Set (SBSET) and the newer Fat Boy Smoothing Blade Set (FSBSET). Each set bundles specific tools aligned with a particular level of automation or finishing need. The Tactical and Predator sets provide comprehensive automatic taping solutions, the Commando set introduces semi‑automatic tools, the Warrior and Sabre sets focus on smoothing blades, and the Fat Boy set combines smoothing blades with additional hand tools and a mud roller.
